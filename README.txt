Sobre la Tarea 6: Curriculum Angular

PREREQUISITOS:
Antes de iniciar, se debe usar npm i para instalar las librerias usadas.

TECNOLOGIAS:
Esta tarea utiliza Angular para funcionar y arrancar en localhost,
aunque solo es un display de datos precargados, por lo que
únicamente utiliza los html y scss que se crearon como
componenetes de Angular.

COMANDOS:
Solo se tiene un comando, siendo este
ng serve --o, que empieza a levantar las páginas y las abre automaticamente
en el navegador de preferencia.
También se puede usar solo ng serve para que se abra sin que redireccione a esa página.

RUTAS:
/ que es el inicio, una breve explicación de lo que se esta viendo
/education que muestra las escuelas que he asistido
/projects que muestra los mejores trabajos realizados respecto a código
/certificactions que muestra los cursos extracurriculares que he tomado
/languages que muestra los lenguajes que he utilizado
/contact que muestra varios métodos de contacto
Todas estas rutas se pueden alcanzar mediante la NavBar que se encuentra en la parte superior de las páginas.
